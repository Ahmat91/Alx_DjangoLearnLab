# LibraryProject - Advanced Django Features

This Django project demonstrates advanced features including:

- Custom User Model (`CustomUser`) with additional fields (`date_of_birth`, `profile_photo`)
- Role-based access control using groups and permissions
- Custom permissions for the `Book` model: `can_view`, `can_create`, `can_edit`, `can_delete`
- Function-based and class-based views for books and libraries
- User authentication: login, logout, registration
- Integration with Django admin for managing users, books, and permissions
